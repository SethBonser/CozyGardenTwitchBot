Commands Reference

Arguments in [brackets] are optional. Arguments in <angle brackets> are required. Slot numbers refer to garden slots (e.g. 1, 2, 3…).

🌿
In Channel Rewards mode (default), actions like getting seeds, watering, and harvesting are done via Twitch channel point rewards — not chat commands. Viewers still use chat commands for planting, viewing the garden, and the shop.
🌸

Viewer Commands

Commands any viewer can type in chat.

🌿 Garden

!garden [slot]
Shows the whole garden and all slot progress. Add a slot number to see detailed info for that slot — plant name, rarity, stage, and water progress.
!seed
Shows the seed you're currently holding. Seeds are gained from channel point rewards (or !buyseed in petals mode).
!plant [slot]
Plants your held seed into the garden. Picks the first empty slot automatically, or specify a slot number to plant there.
Example: !plant 3
!discard
Releases your held seed without planting it. You'll need to get a new one to play again.
!water [slot]
Waters a plant to help it grow. Auto-picks the slot with the lowest water progress if no slot is given. In Channel Rewards mode, watering is done by redeeming the Water Plant reward instead.
Example: !water 2
!harvest [slot]
Harvests a fully bloomed plant and collects 🌸 petals. Part of the payout is shared with everyone active in chat. In Channel Rewards mode, use the Harvest Plant reward instead.
Example: !harvest 1
!petals
Shows your current 🌸 petal balance. Petals are earned by harvesting and shared with active viewers.
!gardeners
Shows the top 3 viewers by waters given — the community's most dedicated gardeners 🥇🥈🥉
!ghelp
Prints a quick command summary in chat — handy for new viewers who want a refresher.

🛒 Shop

!shop
Shows all available items and their petal prices in chat, split into two messages by category.
!buy <item name> [slot]
Generic buy command — accepts any item name (fuzzy matched). You can also use the shorthand commands below instead.
Example: !buytonic 2 or !buy growth tonic 2

🌱 Buy Seeds

!buyseed
Random seed — 60% common, 30% uncommon, 10% rare. costs petals
!buyuncommon
Uncommon seed — 75% uncommon, 25% rare. Never common. costs petals
!buyrare
Rare seed — guaranteed rare plant. costs petals

🌿 Buy Garden Actions

In Channel Rewards mode these are channel point rewards instead. In petals mode they cost petals.

!buywater [slot]
Waters a plant using petals. Auto-picks lowest-progress slot, or target a specific slot. This is an alternate command, use !water instead
Example: !buywater 3
!buyharvest [slot]
Harvests a bloomed plant using petals (harvesting is free — it pays out!). Auto-picks or specify a slot. This is an alternate command, use !harvest instead.
!buyexpand
Adds one new slot to the shared garden. Cost increases quadratically with each expansion.
!buyfertilize <slot>
Applies fertilizer to an empty slot — the next plant placed there needs half as many waters at every stage. petals only
Example: !buyfertilize 2

🧪 Buy Boosts

Single-use consumables — spend petals for an immediate effect.

!buyrain
🌧️ Rain Cloud — instantly waters every occupied garden slot once. Great when the garden is full!
!buytonic <slot>
🧪 Growth Tonic — your next !water on the chosen slot counts as 3 waters instead of 1.
Example: !buytonic 2

🪣 Buy Upgrades

One-time stream-wide purchases — once bought, everyone in chat benefits.

!buycompost
🪣🌿 Compost Bin — all plants need 20% fewer waters per stage. Permanent stream upgrade. 600🌸
!buycopper
🪣 Copper Can — vestigial upgrade (watering cooldown reduction no longer applies). 400🌸
!buysilver
🪣✨ Silver Can — requires Copper Can first. Vestigial upgrade. 800🌸

🌸 Petals Mode Only

!startgarden
Claims your one-time starter petal grant so you can begin spending petals. Only available in petals-only mode — in Channel Rewards mode everyone starts automatically.

Viewer Dashboard

The viewer dashboard is a graphical web app where viewers can check their petal balance, browse the shop, see the live garden, plant their held seed, and review their harvest history — all without typing a single chat command. It's served by the bot and optionally exposed publicly via Cloudflare Tunnel so anyone watching can open it on their phone or laptop.

!dashboard
Posts the dashboard URL to chat so viewers can open it in their browser. The link goes to the public Cloudflare Tunnel URL when DASHBOARD_TUNNEL=true, or to the local URL otherwise. Works in both Channel Rewards and Petals modes.

📋 What's on the dashboard

Stats
Petal balance, held seed with rarity badge, total harvests, and total waters given — the viewer's full garden profile at a glance.
Live Garden
Slot-by-slot view of the shared garden, synced in real time via WebSocket. Each slot shows the plant's pixel-art sprite at its current growth stage, stage label, and water progress. Blooming slots are highlighted in gold.
Plant Your Seed
If the viewer is holding a seed, a panel appears below the garden showing the seed name and rarity. An optional slot input lets them target a specific slot — leaving it blank auto-picks the first empty one. The Plant button posts the result to Twitch chat exactly like !plant.
Shop
Graphical shop organised by category (seeds, garden actions, upgrades, boosts). Viewers can buy items directly — slot-targeting items like Growth Tonic and Fertilize show an inline slot picker. Purchases are processed identically to chat commands and posted to Twitch chat.
Harvest History
Every plant the viewer personally harvested, with the bloom sprite, plant name, rarity, petal payout, and timestamp.
⧉ Popout button
Top-right corner of the nav bar. Opens the dashboard in a compact 430 × 750 popup — great for keeping it open alongside a stream on a second monitor or phone. The username carries over automatically and the popout button hides itself inside the popup.
🔄 Refresh button
Bottom of the nav bar. Re-fetches all data without a full page reload — the typed username is never lost.
🌐
By default the dashboard is local-only (http://localhost:8080/dashboard). Set DASHBOARD_TUNNEL=true in .env and install the cloudflared CLI to expose it publicly. The bot will post the public URL to chat on startup — or viewers can ask for it anytime with !dashboard.
