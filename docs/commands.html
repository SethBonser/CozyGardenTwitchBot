Commands Reference

Arguments in [brackets] are optional. Arguments in <angle brackets> are required. Slot numbers refer to garden slots (e.g. 1, 2, 3…).

🌿
In Channel Rewards mode (default), actions like getting seeds, watering, and harvesting are done via Twitch channel point rewards — not chat commands. Viewers still use chat commands for planting, viewing the garden, and the shop.
🌸

Viewer Commands

Commands any viewer can type in chat.

🌿 Garden

!garden [slot]
Shows the whole garden and all slot progress. Add a slot number to see detailed info for that slot — plant name, rarity, stage, and water progress.
!seed
Shows the seed you're currently holding. Seeds are gained from channel point rewards (or !buyseed in petals mode).
!plant [slot]
Plants your held seed into the garden. Picks the first empty slot automatically, or specify a slot number to plant there.
Example: !plant 3
!discard
Releases your held seed without planting it. You'll need to get a new one to play again.
!water [slot]
Waters a plant to help it grow. Auto-picks the slot with the lowest water progress if no slot is given. In Channel Rewards mode, watering is done by redeeming the Water Plant reward instead.
Example: !water 2
!harvest [slot]
Harvests a fully bloomed plant and collects 🌸 petals. Part of the payout is shared with everyone active in chat. In Channel Rewards mode, use the Harvest Plant reward instead.
Example: !harvest 1
!petals
Shows your current 🌸 petal balance. Petals are earned by harvesting and shared with active viewers.
!gardeners
Shows the top 3 viewers by waters given — the community's most dedicated gardeners 🥇🥈🥉
!ghelp
Prints a quick command summary in chat — handy for new viewers who want a refresher.

🛒 Shop

!shop
Shows all available items and their petal prices in chat, split into two messages by category.
!buy <item name> [slot]
Generic buy command — accepts any item name (fuzzy matched). You can also use the shorthand commands below instead.
Example: !buytonic 2 or !buy growth tonic 2

🌱 Buy Seeds

!buyseed
Random seed — 60% common, 30% uncommon, 10% rare. costs petals
!buyuncommon
Uncommon seed — 75% uncommon, 25% rare. Never common. costs petals
!buyrare
Rare seed — guaranteed rare plant. costs petals

🌿 Buy Garden Actions

In Channel Rewards mode these are channel point rewards instead. In petals mode they cost petals.

!buywater [slot]
Waters a plant using petals. Auto-picks lowest-progress slot, or target a specific slot.
Example: !buywater 3
!buyharvest [slot]
Harvests a bloomed plant using petals (harvesting is free — it pays out!). Auto-picks or specify a slot.
!buyexpand
Adds one new slot to the shared garden. Cost increases quadratically with each expansion.
!buyfertilize <slot>
Applies fertilizer to an empty slot — the next plant placed there needs half as many waters at every stage. petals only
Example: !buyfertilize 2

🧪 Buy Boosts

Single-use consumables — spend petals for an immediate effect.

!buyrain
🌧️ Rain Cloud — instantly waters every occupied garden slot once. Great when the garden is full!
!buytonic <slot>
🧪 Growth Tonic — your next !water on the chosen slot counts as 3 waters instead of 1.
Example: !buytonic 2

🪣 Buy Upgrades

One-time stream-wide purchases — once bought, everyone in chat benefits.

!buycompost
🪣🌿 Compost Bin — all plants need 20% fewer waters per stage. Permanent stream upgrade. 600🌸
!buycopper
🪣 Copper Can — vestigial upgrade (watering cooldown reduction no longer applies). 400🌸
!buysilver
🪣✨ Silver Can — requires Copper Can first. Vestigial upgrade. 800🌸

🌸 Petals Mode Only

!startgarden
Claims your one-time starter petal grant so you can begin spending petals. Only available in petals-only mode — in Channel Rewards mode everyone starts automatically.
